SimilarityJudge
===============
**********************
作业相似度判断系统
V2.0 版本
**********************
使用方法：
1.演示程序目录

2.中的SimiJudge.exe 拷贝到要检测的文件夹下面

3.运行文件，该程序会自动检测当前文件夹下面的 .c .cpp .txt .html 格式的文件
并且进行交叉匹配计算相似度

4.结果会保存到report.txt中
***********************
注意：
1.本程序还在测试阶段，得出的结论仅供参考，如果要使用，请结合人工判断。。

2.相似度等级，与代码的长度有关系，一般来说，代码越长越准确。

如果出现下列等级建议审核一下：
#LEVEL 1#  
#LEVEL 2#  

下面的等级，准确率不高，仅供参考：

#LEVEL 4#
